# Contribution Guidelines

Thank you for considering contributing to the Diyanet-Ezan project! Here are some guidelines to help you get started:

## Reporting Issues
1. **Search for existing issues** before creating a new one. Make sure to check if your issue is already reported.
2. **Provide clear details** when reporting an issue, including steps to reproduce, expected behavior, and actual behavior.

## Contributing Code
1. **Fork the repository** and create your branch from `main`.
2. **Make your changes** and ensure that your code adheres to the existing code style.
3. **Test your changes** thoroughly.
4. **Submit a pull request** describing the changes you made and why they are necessary.

## Code of Conduct
Please adhere to the project's [Code of Conduct](link_to_code_of_conduct) while interacting with others in the community.

## Licensing
By contributing to this project, you agree that your contributions will be licensed under the same license as the project.

Thank you for your contributions!
